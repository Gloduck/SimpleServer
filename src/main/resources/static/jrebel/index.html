激活链接生成

点击复制按钮将激活链接复制到剪贴板，然后在JRebel中粘贴使用

使用说明

注意事项

  • 本工具仅用于开发测试目的
  • 请勿用于商业用途或生产环境
  • 遇到问题请点击查看详细帮助

激活步骤详解

  1. 1

    复制激活链接

    点击"复制链接"按钮，将生成的URL复制到剪贴板

  2. 2

    打开JRebel配置

    在IDE中打开JRebel配置面板，选择激活方式

  3. 3

    选择激活模式

    选择"Team URL"激活方式，粘贴复制的链接

  4. 4

    完成激活

    填写任意邮箱地址，点击激活按钮完成激活

常见问题解决

错误: "Cannot read the array length because 'sigBytes' is null"

解决方案：在JRebel激活界面先点击"离线"模式，然后再次点击"在线"模式即可解决

激活链接无效

解决方案：点击刷新按钮生成新的链接，然后重新复制尝试
